07

Do's & Don'ts

The rules that keep the system from drifting back into the AI palette over time.

✓ Do

  • Use semantic tokens — never raw hex or raw oklch in a component.
  • Reserve oxblood for one primary action per region.
  • Use teal only for verified evidence.
  • Keep shadows neutral; let hairline borders structure layout.
  • Write sentence-case copy with plain verbs.
  • Test text contrast in both themes (AA min, AAA body where feasible).

✗ Don't

  • No blue, purple, indigo, violet, fuchsia, or pink — the AI palette.
  • No color-tinted glow shadows (purple card glows banned).
  • No gradients on text or heroes.
  • No rounded-full on cards, inputs, or panels.
  • No warm beige neutrals; canvas is near-pure white/black.
  • No Inter as display; no emoji as icons (inline SVG, currentColor).
